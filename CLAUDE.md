# CLAUDE.md

This is "Between the Prompts", a Hugo blog styled with TailwindCSS focused on AI tools in software engineering, hosted at betweentheprompts.com.

## Development Commands

```bash
npm install          # setup
npm run dev          # TailwindCSS watch + Hugo server (http://localhost:1313)
npm run build        # production build → public/
```

## Content

- Blog posts are Markdown files in `content/`
- Hugo version pinned to 0.128.0 in the GitHub Actions workflow

## Git Workflow

Always rebase, never merge.
